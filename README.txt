This is a github repository for homology modeling of the RANBP2- HIV/SIV system by The Wilke Lab (http://wilkelab.org/).

scorefiles/
    The raw scorefiles from RosettaDock of the virus-RANBP2 complexes

cleaned_structures/
    Structures of the original complex used for modeling
